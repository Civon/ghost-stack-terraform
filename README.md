# IaaS for GhostCMS

## Prerequisites

- terrafrom
- export ARM credentials to environment via cloudshell/bash(credentials see notion)

## Stack(TBD)

- Cloudflare(Domain)
- Cloudfront(CDN)
- Nginx(Cache)
- Ghost
- MySQL
- Azure Storage

## Usage

VS Code command plate:

1. Terraform: push
2. Terraform: init
3. Terraform: plan
4. Terraform: apply
5. if needed, Terraform: destory
